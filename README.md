A set of security tools for Django.

# Install the package:

`pip install django-security-tools`

# Add the app to INSTALLED_APPS:

```
INSTALLED_APPS = [
    # Other apps...
    'djangosecuritytools',
]
```

# Apply migrations:

`python manage.py migrate`


# Install in dev

`pip install -e path/to/django-security-tools`
